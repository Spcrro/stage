Guide
  • Minecraft est le jeu le plus vendu

    • Minecraft java est le jeu le plus vendu avec 350 millions de ventes
    (le titre est parfois donné a Tetris, cependant, Tetris n'a plus de vente que Minecraft
    que si on considère toute les version de Tetris comme un seul et même jeu.)

  • il existe beacoup de version de Minecraft
    • Minecraft JAVA : la version originale sur PC.
    • Minecraft Bedrock: la version multiplateforme, plus optimisée mais moins libre.
    • la version "legacy": l'ancienne version console de Minecraft developpée par 4j studios.
    • la version classique: la première version de Minecraft disponible gratuitement dans le navigateur.
    • et bien d'autres souvant basées sur la version Bedrock comme la version "éducation" ou la version VR (qui a été abandonnée)
      mais également des versions "non-officiels" comme pojav (ou pocket java) qui est un portage de la version java sur android.

  • Il est possible de faire tourner Minecraft dans Minecraft

    • grâce au mod "VM computers", il est possible de faire tourner des machines virtuelles depuis Minecraft.
    donc, en paramétrant cette machine au préalable, il est possible de lancer et de jouer a Minecraft dans Minecraft.

  • Nouvelle mise a jour

    • Une nouvelle mise a jour de minecraft est sortie le mardi 17 juin, rajoutant notament les ghasts gentils.
    Ce sont des ghasts que vous pouvez apprivoiser pour les monter et voyager dans le ciel.
    Cette mise a jour rajoute également les "graphismes éclatants", uniquement disponible (bour Bedrock pour le moment)
    modifiant les graphismes du jeu pour les rendre similaire au shaders de la version java.
